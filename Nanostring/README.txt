NANOSTRING README FILE




This folder contains all code and excel files related to the data processing and analysis of the R35 Nanostring data




!!!!!!!!!!!!!!!!!!!!!!!!!!!!
R35 Nanostring Data Analysis
!!!!!!!!!!!!!!!!!!!!!!!!!!!!

- Contains all the R code and excel tables related to the summary statistics and differential gene expression analysis


!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
R35 Nanostring Data Preprocessing
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

- Contains all the R code and excel tables related to the preprocessing of the R35 Nanostring data. This includes the modification of metadata, the loading and combining of RCC files, quality control, normalization, and creating data tables with metadata markers


!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
R35 Nanostring Data Visualization
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

- Contains all R code and excel tables related to the visualization of R35 Nanostring data (such as individual gene comparisons and total group comparisons). These include box plots, volcano plots, heat maps, histograms, etc. 


!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
R35 Nanostring Original data
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!


- Contains all the original data used for Nanostring data analysis in this project. These include the original unmodified metadata, the RCC files, etc.
